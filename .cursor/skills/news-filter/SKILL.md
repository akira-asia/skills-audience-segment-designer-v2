# News Filter

## 事前準備（重要）

分析開始前に Web 検索または信頼できる API で最新情報の取得を試みる。ポートフォリオ銘柄は `data/portfolio.csv` 等、ユーザーが指定するファイルがあれば必ず参照する。

## 分類基準

### Tier 1: 即座に対応

- 保有銘柄の決算（コンセンサスとの乖離が大きい場合など）
- 保有銘柄に関する規制・訴訟
- 主要なマクロイベント（政策金利、地政学リスク）

### Tier 2: 当日中に確認

- 経営陣変更、大型投資、社債発行
- 競合の重大発表

### Tier 3: 週次レビューで確認

- アナリストレーティング変更
- 業界レポート

### Tier 4: 低優先（ノイズに近い）

- 短期的な株価変動のみの記事
- 根拠の薄い予測・クリックベイト

## 出力

- Tier ごとに箇条書き
- 各項目に「なぜその Tier か」を 1 行
- 保有銘柄に関係がなければその旨を明示
