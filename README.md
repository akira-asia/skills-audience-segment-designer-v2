# skills-audience-segment-designer-v2

ウェブ広告向けのオーディエンスセグメント設計を支援する Skill（`SKILL.md`）と、Cursor 用 Agent Skills（`.cursor/skills/`）をまとめたリポジトリです。

## 入力について（必読）

本 Skill を最大限に活かすには、**あらかじめ実施した分析の成果物を Markdown で入力**してください。

1. **ビジネス分析**を実施したうえで出力した `.md`
2. **5seg 分析**を実施したうえで出力した `.md`

5seg に関しては、上記に**厳密に準拠した md でなくても構いません**。5seg に沿った課題解決に紐づく**提案資料**（分析・提案の内容が分かるもの）を入力いただいても問題ありません。

手法・テンプレートの参照先として、[Method Library（フルスタックマーケター育成講座）](https://method-library.kandoasia.com/) を利用できます。

## リポジトリ構成（概要）

| パス | 内容 |
|------|------|
| `SKILL.md` | オーディエンスセグメント設計のメイン定義 |
| `references/` | `SKILL.md` から参照する補助ドキュメント（シンボリックリンク） |
| `.cursor/skills/` | Cursor Agent 向けスキル（コードレビュー、投資ワークフロー等） |

## ライセンス・免責

リポジトリの利用条件や免責は、必要に応じて別途整備してください。投資関連のスキルは分析・整理用途のテンプレートであり、投資勧誘ではありません。
