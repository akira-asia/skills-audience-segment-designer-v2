# Stock Valuation Checker

## 事前準備（重要）

株価・財務指標は、利用可能な API または信頼できる一次・二次ソースで取得した最新値を使う。

## 評価手法

### 1. 相対評価

- PER: 過去レンジ、同業比較（分かる範囲で）
- PBR、EV/EBITDA: 同様

### 2. DCF（ディスカウントキャッシュフロー）

- 前提: 成長率（楽観/中立/悲観）、割引率、ターミナルバリュー
- 前提は必ず明示

### 3. 感応度

- 成長率の変化が評価に与える影響
- 割引率の変化の影響

### 4. テクニカル（参考）

- 52 週レンジ内の位置、主要移動平均からの乖離（参考）

## 禁止事項
