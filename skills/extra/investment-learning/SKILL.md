# Investment Learning

## 抽出対象

### スキル化しやすいもの

- 再現性のある分析手順（例: 決算ではセグメント売上→ガイダンス乖離の順で見る）
- 市場・セクター固有の注意点
- 感情に流されないためのルール

### スキル化しにくいもの

- 銘柄一回限りの細部（ノートに残す程度）

## スキル化の判断基準

以下を満たすものを優先:

- 非自明で、汎用の本だけでは足りない
- 同じ状況で再び使える
- 実体験で有効だった（または検証可能）

## 手順

1. セッションで得た学びを列挙
2. 上記基準でフィルタ
3. 新規スキル名（ケバブケース）または既存スキルの追記案を提示
4. ユーザー確認後、`SKILL.md` または `references/` への追記案を出す
