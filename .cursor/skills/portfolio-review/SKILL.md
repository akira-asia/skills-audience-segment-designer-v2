# Portfolio Review

## 入力

- `data/portfolio.csv` またはユーザーが示す銘柄・比率・コスト情報

## チェック項目

### 1. 集中度リスク

- 単一銘柄のウェイト上限の超過の有無
- セクター集中度

### 2. 相関リスク

- 同一テーマへの重複露出
- マクロ要因への感応（分かる範囲で）

### 3. 個別銘柄

- 投資テーゼの要約と現状の整合
- 目標・損切りラインへの距離（ユーザーが設定している場合）

## アラート例（ユーザー設定に合わせて調整）

- 単一銘柄が高ウェイト
- セクターが偏りすぎ
- 複数銘柄がリスクラインに接近

## 出力

- 優先度の高い懸念から列挙
- 次に取るべき確認アクション（データ取得・ニュース・決算日程等）
