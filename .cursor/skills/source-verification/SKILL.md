# Source Verification

## 検証プロセス

### 1. 一次ソースの特定

- SEC Filing（10-K、10-Q、8-K、6-K 等）
- 公式プレスリリース
- 決算説明会トランスクリプト

### 2. 二次ソースとの照合

- 複数の報道で数値・主張が一致するか
- 引用元が正しいか

### 3. 文脈の確認

- 引用が原文と一致するか
- 省略された文脈で意味が変わっていないか

## 出力形式

- 検証結果: 確認済み / 一部確認 / 未確認
- 一次ソースへのリンクまたは参照元
- 乖離がある場合は内容

## 禁止事項

- 未検証の数値を「事実」として断定しない
