# Stop-Loss Decision

## 判断フレームワーク

### 1. テーゼの検証

- 投資時の前提は今も有効か
- 成長ドライバー・競争優位は維持されているか
- 経営陣への信頼に重大な変化はないか

テーゼが崩れている場合は、価格以前に売却を検討。

### 2. 定量基準（例・ユーザー設定で置き換え）

- 購入価格から一定割合以上の下落
- 52 週高値からの下落幅
- ベンチマークに対する相対パフォーマンス

### 3. 機会コスト

- 同額を別の選択に振った場合の比較（概念レベルでよい）

## 出力形式

- 結論: 売却検討 / 保有継続 / 要監視
- 判断根拠: 上記フレームワークに沿って簡潔に
- 次のアクション: 具体的な確認事項

## 禁止事項

- 個人の投資を断定して勧誘しない（整理と論点提示に留める）
