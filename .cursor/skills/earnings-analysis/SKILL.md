# Earnings Analysis

## 事前準備（重要）

- 最新の開示資料（8-K / 6-K / 10-Q / 10-K 等）と決算資料
- 可能ならカンファレンスコール要約

## 分析フレームワーク

### 1. トップライン

- 売上高: 前年同期比、前四半期比、コンセンサス比（分かる範囲で）
- セグメント別売上: 成長ドライバー
- 地域別: 地理的リスク

### 2. 収益性

- 粗利益率・営業利益率・純利益率のトレンド
- 一時的要因の有無

### 3. キャッシュフロー

- 営業 CF、設備投資、フリー CF

### 4. ガイダンス

- 次期ガイダンスとコンセンサス（分かる場合）
- 前回からの修正

### 5. カンファレンスコール

- 経営陣が強調した点
- アナリスト質問の傾向

## 出力の分離

- **Fact**: 検証可能な数値・開示
- **Guidance**: 経営陣の公式見通し
- **Speculation**: 解釈・推測は「推測」と明記
