# Analysis Writer

## 記事構成

### 1. 導入

- 具体的な数字や出来事から入る
- 読者の関心を引く 1〜2 文

### 2. 本文

- **Fact**: 検証可能なデータ。出典を明記
- **Guidance**: 経営陣が公表した見通し
- **Speculation**: 分析者の解釈。「推測」と明記

### 3. 結論

- 要点の再確認
- 今後の注目点・リスク

## 禁止事項

- 「まず〜、次に〜、そして〜」だけの機械的接続に頼らない
- 未検証の数値を事実として書かない
- リスクを省略しない
